Blessed Maximum Care AFH — Netlify Upload
=========================================

Files included:
- index.html (blue/gray theme + gallery uploads + Netlify Form + reCAPTCHA + map)
- netlify.toml (redirects for form success + SPA fallback)

Deploy steps (drag-and-drop):
1) Visit https://app.netlify.com and log in.
2) Click "Add new site" → "Deploy manually".
3) Drag this ZIP into the upload area.
4) Netlify will give you a public URL instantly.

Turn on email notifications:
- Netlify Dashboard → Forms → contact → Notifications → Add notification → Email → blessedmaxcare@gmail.com

Add a custom domain (optional):
- Site settings → Domains → Add custom domain → enter your domain and follow the DNS instructions.
